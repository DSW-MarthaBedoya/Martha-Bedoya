Proposito del sitio

Es este espacio encontraras, las enseñanzas para dinamizar el aprendizaje de las temáticas básicas de: Etiquetas Básicas de HTML, Atributos y Valores, los cuales permiten conformar una página web, con elementos primordiales tales como: párrafos, imágenes, viñetas, entre otros, a través de estas síntesis se conforma la página web de acuerdo con las necesidades y requerimientos del usuario.

Tema 1: H2 Etiqueta Básica de HTML

HTML: (Lenguaje de Marcas de Hipertexto, del inglés - Hyper Text Markup Language). Este el componente más básico de la web.

Es un lenguaje artificial que los computadores, son capaces de interpretar, y diseñado para que los programadores transcriban instrucciones que los navegadores llevan a acabo para originar la pagina web. Por lo anterior se puede decir que HTML, es un lenguaje de programación o “idioma que la maquina entiende y procesa para dar una respuesta”.

A continuacion se nombran algunas etiquetas basicas: existen para, contenido, informacion sobre el documento, división dentro del contenido, enlaces, para colocar textoen negrita, saltos de linea, titulos dentro del contenido, añadir imaganes al documento, listas ordenadas, listas desordenadas, elementos dentro de la lista, parrafos, estilos de una parte del texto.

Es un lenguaje de programación básico pero imprescindible para los programadores. Su funcionamiento se basa en el uso de etiquetas para dar las instrucciones necesarias al navegador y que este las interprete como debe mostrar los elementos en la página. El código HTML es escrito por el programador con un editor de texto, y este a su vez se guarda como un documento de extensión *.html, lo que conlleva a que el archivo se pueda abrir en el navegador.

Tema 2: Atributos y Valores

El Atributo, siempre se deben encontrar en la etiqueta inicial, son palabras especiales ubicadas dentro de esta etiqueta.

Los atributos son elementos de las etiquetas que pueden proporcionar ciertas características a un texto, imagen etc. (altura, ancho, color). Van insertos dentro de la etiqueta de inicio o apertura los cuales llevan detrás del signo igual (“=”) los valores del atributo.

HTML. No distingue las mayúsculas de las minúsculas, lo que hace que sea necesario cerrar entre comillas dobles.

Valores: los valores son aquellos que siempre debe permanecer encerrados en comillas, ya sean simples o dobles. ej. “imagen.png” Por lo anterior se dice que el Atributo y Valor deben estar unidos por el signo: =
